 
## Dateiinhaltsformat
- *_prompt.txt- Audio-Tags

Einfache, durch Kommas getrennte Audiomerkmale, die den Klang, die Instrumente, das Genre, die Stimmung usw. beschreiben.

** Richtlinien zum Erstellen von Eingabeaufforderungs-Tags:
- Geben Sie das Genre an (z. B. "boom bap", "trap", "gangsta")
- Geben Sie 2 Stimmtypen an (z. B. "sad", "deep", "energetic", "hard-hitting")
- Geben Sie 3 tatsächlich gehörte Instrumente an (z. B. "sub bass", "piano", "synth", "drums")
- Geben Sie den Rap Style an (z. B. "energetic rap", "lyrical rap", "aggressive rap", "melodic rap")
- Geben Sie die Vocal Typen an, falls bekannt (z. B. "male rap", "female singing", "male feature rap")
- Geben Sie den Rap Flow an , falls bekannt (z. B. "normal flow", "double time flow", "chopper flow")
- Geben Sie die Vocal Fx an , falls bekannt (z. B. "clean vocals", "auto-tune vocals", "Harmony Vocals")

** Beispiel:

- boom bap, deep, smooth, hard drums, piano, strings, storytelling rap, male rap, male feature rap, normal flow, echo vocals

- Alle Tags in Englischer Sprache
